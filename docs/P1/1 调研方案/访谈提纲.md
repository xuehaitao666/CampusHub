# 用户访谈提纲

## 一、 行为与场景挖掘
* **Q1：最近一次经历**
  你最近一次需要别人帮忙（或者帮别人做事）是什么情况？能具体讲一下吗？
* **Q2：困难与痛点**
  在这个过程中，你遇到过哪些困难或不方便的地方？
* **Q3：现有解决方案**
  你当时是怎么解决这个问题的？效果怎么样？

## 二、 意愿与动机探索
* **Q4：使用契机**
  如果有一个平台可以帮你快速找到人或接任务，你觉得在什么情况下你会使用它？
* **Q5：角色偏好**
  你会更愿意发布需求，还是接单？为什么？
* **Q6：风险顾虑**
  在找人帮忙或接单的过程中，你最担心的是什么？

## 三、 功能需求与核心价值
* **Q7：理想功能**
  如果这个平台存在，你最希望它具备哪些功能？
* **Q8：核心价值（MVP）**
  如果只能保留一个功能，你觉得最重要的是哪个？为什么？
