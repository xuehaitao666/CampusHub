# CampusHub 校园互助平台：大学生使用意愿及需求调查问卷

1. **给出你希望有的板块**
2. **请问你认为互助平台应该包含哪些功能？**
3. **以上功能或板块中，你认为最重要的是？**
4. **你认为通过学号认证注册方式如何？**
5. **为了让平台活下去且维持良好氛围，你认为需要管理哪些方面？**
6. **你希望平台的收费方式是？**
7. **如果平台需要志愿管理员，你的应聘意愿是？**
8. **你对平台建设还有哪些建议？**
9. **要是平台真建成了且真投入使用了，你的使用意愿是？**
10. **你在生活中还有什么需要他人参与协助的需求？**
11. **你在互助平台上想要得到怎样的酬劳，你愿意接受本平台的积分吗？**
12. **在互助的过程中，你认为需要与对方进行线上交流吗？**
13. **你认为平台应该通过什么标准来判断一个互助任务成功或者失败？**
